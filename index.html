PhotoShare

Frictionless Photo Hosting & Sharing

Sharing photos with friends, family and socials is not easy, especially when transferring between different ecosystems like Windows and ios. Previously, whenever i wanted to share a photo from my dslr, I would upload the photo to my laptop, edit with photoshop, upload to google drive, download to my phone, and finally share the photos.

So I built a direct publishing pipeline taking advantage of GitHub's cloud storage and static hosting. I made a custom photo editor which processes source media and publishes directly to GitHub Pages. Zero-cost, full-quality distribution with global CDN delivery, universal device access and indefinite storage backup.

Tech Stack:

  • HTML5, CSS3, JavaScript
  • GitHub Pages, Static Hosting
  • Custom Photo Editor Integration
